## 📖 О проекте

TaskFlow - это **легковесный**, _быстрый_ и _**удобный**_ инструмент для работы с задачами.

> Наша миссия - сделать управление задачами максимально простым и доступным.

Проект поддерживает работу через командную строку и хранит данные локально[^1].

---

## 🚀 Возможности

- Добавление задач
- Удаление задач
- Отметка выполнения
    - Быстрое переключение статуса
    - Фильтрация выполненных задач
- Сортировка по приоритету

## ⚙️ Установка

1. Склонируйте репозиторий:
```bash
git clone https://github.com/example/taskflow.git
```
    
2. Перейдите в папку проекта:
```bash
cd taskflow
```
    
3. Установите зависимости:
```bash
pip install -r requirements.txt
```
    

---

## 💻 Пример использования

```python
from taskflow import TaskManager  
  
manager = TaskManager()  
manager.add_task("Сделать практическую работу")  
manager.complete_task(1)

```

Использование команды `task add "текст"` добавит новую задачу.

---

## 📊 Таблица команд

| Команда | Описание              | Пример          |
| ------- | --------------------- | --------------- |
| add     | Добавить задачу       | task add "дело" |
| done    | Завершить задачу      | task done 1     |
| list    | Показать список задач | task list       |
